# Simple Log Analyzer

## Project Description
Simple Log Analyzer is a Python project that analyzes log messages and counts different types of logs such as ERROR, INFO, and WARNING. The program also finds the most frequent log type while ignoring case sensitivity.


## Features
- Count ERROR logs
- Count INFO logs
- Count WARNING logs
- Find most frequent log type
- Ignore case sensitivity
- Pattern detection using string processing


## Concepts Used
- Lists
- Loops
- Conditional Statements
- Dictionaries
- String Processing
